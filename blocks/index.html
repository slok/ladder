Scalers
Dummy Dummy scaler will return changed or not if the current quantity is same/different of the new quantity, if different it will be the new current Name dummy Options: wait_duration: The time to wait when waiting after scaling, by default 0ms Example scale: kind: dummy config: wait_duration: 5s Warning Only used for testing Stdout Dummy scaler will return changed or not if the current quantity is same/different of the new quantity, if different it will be the new current and will put a message with the action made (upscale or downscale)

Filters
Dummy Dummy filterer will return a concrete value Name dummy Options filters: - kind: dummy config: Warning Only used for testing Limit Limit filterer will restrict the scalign value to the max and min configured limits Name limit Options max: The max value min: The min value Example filters: - kind: limit config: max: 10 min: 2 ECS running tasks ECS running tasks filter will check how many tasks are pending by the ECS scheduler on a given cluster, if this number exceeds the desired one then it will break the filters chain and set the autoscaling to the current number.

Solvers
Note You only will need a solver if you have multiple inputters, if not it will be ignored although there is configured on the file Dummy Dummy solver will return the sum of all the inputs Name dummy Options No options Example solve: kind: dummy Warning Only used for testing Bound Bound solver will return the max or min of all the received inputters quantity. Name bound

Arrangers
Dummy Dummy gatherer always will return a constant quantity Name dummy Options quantity: The quantity to return always Example arrange: kind: dummy config: quantity: 0 Warning Only used for testing In list In list arranger will arrange a new quantity to scale up/down based on two lists When the quantity received is in match_upscale list it will scale the current quantity by the match_up_magnitude percent. The same for the downscale.

Gatherers
Dummy Dummy gatherer always will return a constant quantity Name dummy Options quantity: The quantity to return always Example gather: kind: dummy config: quantity: 0 Warning Only used for testing Random Random gatherer will return a random number between a max and min bounds Name random Options max_limit: The max limit of the random (not included) min_limit: The min limit of the random (included) Example gather: kind: random config: max_limit: 10 min_limit: 0 SQS property SQS gatherer will return the quantity of the number of messages of a queue, this number can be one of the available SQS prperties
